
Projet INF203 en L1 informatique UGA : Projet cowsay.
INF203 project, L! COmputer Science at UGA : Projet cowsay.

--
Projet illustrant la transition du scripting Bash (automatisation et calculs) vers la programmation système en C (gestion d'arguments et animations).
Le dépôt inclut une réimplémentation partielle de l'utilitaire Cowsay et un simulateur Tamagoshi interactif gérant le cycle de vie et la santé d'une vache en temps réel.
--
This project illustrates the transition from Bash scripting (automation and calculus) to C system programming (argument handling and animations). The repository includes a partial reimplementation of the cowsay utility and an interactive Tamagoshi simulator that manages a cow's life cycle and health in real-time.
